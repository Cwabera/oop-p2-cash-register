# Cash Register Lab

This project implements a `CashRegister` class for an e-commerce site.

## Features

- Add items to cart
- Track total price
- Count items in cart
- Apply a percentage discount
- Void the last transaction

## Project Structure

- `lib/cash_register.py` — main class implementation
- `testing/` — test files

## Run Tests

Using pipenv:

```bash
pipenv install
pipenv run pytest
